tweetgrab
=========

Grabbing Twitter Information


Requirements
------------

- python ``twitter`` module
  ::

      pip install twitter



Usage
-----

::

    $ python tweetgrab.py
    Use OAuth (y/n)? y
    Target user screen name : saktidc
    fetching followers
    saving relationship to file
    Done.
    TweetGrab for saktidc, remaining limit 346

    $ view saktidc_rel.csv
